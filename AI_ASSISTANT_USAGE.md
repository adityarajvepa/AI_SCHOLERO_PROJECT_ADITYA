# AI_ASSISTANT_USAGE

I used AI tools during this project, mainly ChatGPT and Cursor.

## Where I used AI assistance

I used ChatGPT mostly to help me think through the project at a systems level. The biggest help was in coming up with prompts that I could give to Cursor so it could build parts of the project end to end. I also used ChatGPT to help me come up with demo questions, review whether certain answers looked grounded or weak, and understand what parts of the system still needed tuning.

I used Cursor heavily for implementation. Most of the actual coding work was done through Cursor agents based on the prompts and directions I gave. This included building and modifying the file structure, backend routes, UI changes, retrieval flow, quiz flow, and general project wiring.

## What AI helped accelerate

AI tools helped me move faster on:
- planning the project structure
- iterating on backend and UI changes
- debugging issues
- testing ideas for retrieval and answer quality
- drafting and refining documentation
- deciding what kinds of questions to use for demos and validation

ChatGPT was especially useful when I got stuck and needed help reasoning through what was going wrong, what should be changed next, or how to explain the system more clearly.

Cursor was useful for quickly turning those ideas into code changes across multiple files.

## External model/API usage

For answer generation and quiz generation, I used the Gemini API because it was easy to integrate into this project and worked well for the take-home scope. The retrieval, ingestion, and application logic were still part of the system I built around it.
